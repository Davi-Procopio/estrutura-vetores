# 📌 Estrutura de Vetores

## 📖 Descrição
Este repositório contém atividades práticas desenvolvidas em JavaScript com foco na utilização de vetores (arrays), manipulação de dados e uso de estruturas de repetição para percorrer informações armazenadas.

---

## 🎯 Objetivo
- Praticar a criação e manipulação de vetores
- Armazenar múltiplos valores em arrays
- Utilizar estruturas de repetição (`for`) para percorrer vetores
- Realizar operações como:
  - soma de números pares
  - geração de números aleatórios
  - contagem de ocorrências
  - remoção de valores duplicados
- Utilizar versionamento com Git e GitHub

---

## 🛠️ Tecnologias Utilizadas
- JavaScript
- Visual Studio Code
- Node.js
- Git e GitHub

---

## 📂 Estrutura do Projeto
O projeto contém 4 algoritmos em JavaScript:

1. Soma dos números pares armazenados em um vetor  
2. Geração automática de números aleatórios em vetor  
3. Contagem de quantas vezes um número aparece no vetor  
4. Criação de um vetor sem valores duplicados  

---

## ✅ Resultados Esperados
- Vetores declarados e manipulados corretamente
- Utilização de laços de repetição para percorrer arrays
- Leitura e armazenamento de dados em vetores
- Programas funcionando corretamente após testes
- Código organizado e versionado no GitHub
